# 信息采集问题序列

当用户触发 `/heartbeat-review` 或 `/heartbeat-track` 后，依次询问以下问题（最多 4 个）。用户可以跳过任何可选项。

---

## 问题序列

**Q1（必填）：双方昵称**

```
请告诉我你们的昵称：
- 你叫什么？（例如：小明 / 我 / Ming）
- TA 叫什么？（例如：小红 / 她 / Amy）
```

**Q2（必填）：基本背景**

```
一句话描述关系背景：
例如：在一起两年，大学同学，分手半年 / 暗恋阶段，认识3个月，还没表白
```

**Q3（必填）：选择分析模式**

```
你想用哪个模式？

  [A] 复盘型 — 已经分手/结束，想回顾整段关系的情感变化
  [B] 追踪型 — 关系还在进行中，想持续追踪你们的情感走势

选 A 或 B：
```

**Q4（可选）：补充说明**

```
有没有想特别关注的时间节点？（可跳过）
例如：2024年3月我们第一次吵架 / 五月她突然开始冷淡

如果有，告诉我大概时间和发生了什么，我会在曲线上标注出来。
```

---

收完信息后汇总确认，再进入原材料导入环节。
