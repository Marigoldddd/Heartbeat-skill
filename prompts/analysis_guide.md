# 分析维度指南

当 Claude 收到 `sentiment_scorer.py` 的评分 JSON 后，按以下框架进行解读和标注，
用于在 report_writer 输出的报告中补充「AI 洞察」部分，以及在曲线图的 `events` 字段填入事件标签。

---

## 一、关系阶段识别

根据双方平均好感度，将时间段归入以下阶段：

| 阶段 | 分值范围 | 典型特征 |
|------|----------|----------|
| 热恋期 | 70–100 | 频繁联系、情感词密集、双方都很主动 |
| 稳定期 | 55–69 | 互动规律、情感平稳、偶有起伏 |
| 平淡期 | 40–54 | 例行联系、新鲜感减退、消息变短 |
| 衰退期 | 20–39 | 主动性下降、回复变慢、情感词减少 |
| 冰点期 | 0–19 | 几乎无互动、或仅有冷漠例行消息 |

---

## 二、常见好感度下滑模式

### 1. 渐行渐远型
- 特征：双方好感度缓慢同步下滑，没有明显拐点
- 原因：无重大冲突，但生活轨迹分离、共同话题减少
- 曲线形态：两条线平行地缓慢下沉

### 2. 冷战拉锯型
- 特征：出现锯齿状波动，下滑后有短暂拉升再下滑
- 原因：反复争吵-和好-再争吵的循环
- 识别信号：负面情感词（生气/吵架/委屈）密集后跟着正向词（和好/抱歉/还爱你）

### 3. 单方付出型
- 特征：一条线持续高位，另一条线持续低位且下滑
- 原因：一方持续主动，另一方情感已在退出
- 识别信号：主动性差距 > 30%，回复速度差异显著

### 4. 突发断崖型
- 特征：某个时间点后好感度骤降
- 原因：重大事件（分手争吵/出轨/长期分离）
- 识别信号：单个窗口内分值跌幅 > 20

### 5. 单向升温型
- 特征：一方升温，另一方平稳或下滑，产生错位
- 原因：暗恋未果、或一方在追求另一方
- 识别信号：me_initiative 显著高于 them_initiative

---

## 三、关键转折点提炼原则

从评分数据中寻找以下类型的节点，并在报告中重点标注：

1. **单窗口跌幅 ≥ 15**：可能发生了重要冲突或转变
2. **连续 3 个窗口下滑**：趋势性衰退开始
3. **好感度差值突然扩大（>20）**：情感错位出现
4. **短暂反弹后继续下滑**：典型冷战-和好-再恶化结构
5. **历史最高点**：关系最好的时候，值得在报告中点名

---

## 四、如何填写 events 字段

`events` 是 scorer 输出 JSON 中每个窗口的数组，初始为空，由 AI 根据以下规则填写：

- **用户提供了时间节点信息**：直接填入对应窗口，例如 `["首次争吵"]`
- **检测到骤降**：填 `["⚠️ 情感骤降"]`
- **检测到历史峰值**：填 `["🔆 关系最佳期"]`
- **检测到连续 3 期下滑的起点**：填 `["开始衰退"]`
- **检测到显著反弹**：填 `["💫 短暂升温"]`

格式要简短（≤6个字），因为要显示在图表上。

## 五、追踪模式下的趋势预测语言

| 趋势组合 | 建议措辞 |
|----------|---------|
| 双方↑ | 关系升温中，建议保持当前互动节奏 |
| 我↑ TA↓ | 情感出现错位，TA 的投入在减少，需要关注 |
| 我↓ TA↑ | 对方在升温而你在退出，留意自己的情感状态 |
| 双方↓ | 关系进入降温阶段，可考虑主动创造新的连接点 |
| 双方→ | 关系趋于平稳，没有明显信号 |
